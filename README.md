# java-web-maven-spring-jsp-hello-world

## Description
A POC for springboot web app with jsp support.

## Tech stack
- java
- maven
  - springboot
  - jsp
  - bootstrap

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
[Available at](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
